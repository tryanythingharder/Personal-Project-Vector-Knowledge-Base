## [0.1.0] - 2026-05-22

### Added

- Desktop-first knowledge base application structure with Electron, React, and FastAPI.
- Document ingestion, folder import, and knowledge base indexing workflow.
- RAG chat with citations, evidence preview, and conversation history.
- Model management with multi-provider support and chat model switching.
- Remote backend mode and Docker deployment files.
- Team collaboration basics including invitations, workspace roles, and project sharing.
- Usage tracking, evaluation workbench, and project memory workflows.
- Public repository documentation, MIT license, and contributor guidelines.

### Changed

- Refined the product UI toward a cleaner, lighter, more app-like workflow.
- Simplified and standardized the README structure for open source publication.

### Notes

- The project is still evolving quickly, so internal APIs and deployment details may continue to change in future releases.
